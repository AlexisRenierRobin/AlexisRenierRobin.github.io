Detailed Description

Class which contains the Lattice Boltzmann algorithm used for dithering.

The class relies on other class for the implementation of some computations (computation of equilibrium state and type of microscopic state).

void LBMDither::Collide ( )
private

Does the collide step of Lattice Boltzmann method.

The step is :

\begin{align*} f'_{ij}(x,t+1) = (1-\omega) * f_{i}(x,t) + \omega * f_{i}^{eq}(x,t) \end{align*}

◆ PrepareImage()

void LBMDither::PrepareImage ( )
private

Prepares the image.

Image preparation consists of modifying the image so that :

\begin{align*} \left( \sum_{image \; pixels}^{} ImagePixel \right) \; \% \; 255 \; &= \; 0 \end{align*}

This is done by changing the value of nearby pixels from 255 to 255
